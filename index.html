O .NET Framework (pronuncia-se dot net) é um framework de software desenvolvido pela Microsoft que é executado principalmente no Microsoft Windows. Ele inclui uma biblioteca de classes grande conhecida como FCL (Framework Class Library) e fornece interoperabilidade de idioma (cada idioma pode usar código escrito em outros idiomas) em várias linguagens de programação. Os programas criados para o .NET Framework são executados em um ambiente de software (em contraste com o ambiente de hardware), conhecido como Common Language Runtime (CLR), uma máquina virtual de aplicativos que fornece serviços como segurança, gerenciamento de memória e tratamento de exceções. FCL e CLR juntos constituem o .NET Framework.

O Microsoft SQL Server é um sistema de gerenciamento de banco de dados relacional desenvolvido pela Microsoft. Como um banco de dados, é um produto de software cuja função principal é armazenar e recuperar dados conforme solicitado por outros aplicativos de software, sejam eles no mesmo computador ou aqueles executados em outro computador em uma rede (incluindo a Internet). Há pelo menos uma dúzia de edições diferentes do Microsoft SQL Server destinadas a públicos diferentes e para cargas de trabalho que variam de pequenos aplicativos de uma única máquina a grandes aplicativos voltados para a Internet com muitos usuários simultâneos. Suas principais linguagens de consulta são o T-SQL e o ANSI SQL.

O MySQL é (em março de 2014) o segundo sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto mais utilizado no mundo. É nomeado após a filha do co-fundador Michael Widenius, My. A frase SQL significa Linguagem de Consulta Estruturada.

JavaScript (JS) é uma linguagem de programação dinâmica. É mais comumente usado como parte de navegadores da Web, cujas implementações permitem que os scripts do lado do cliente interajam com o usuário, controlem o navegador, se comuniquem de maneira assíncrona e alterem o conteúdo do documento exibido. Ele também está sendo usado em programação de rede do lado do servidor (com Node.js), desenvolvimento de jogos e criação de aplicativos de desktop e móveis.

Bootstrap é uma coleção gratuita de ferramentas para criar sites e aplicativos da web. Ele contém modelos de design baseados em HTML e CSS para tipografia, formulários, botões, navegação e outros componentes de interface, além de extensões JavaScript opcionais. Em junho de 2014, foi o projeto número 1 no GitHub, com mais de 69.000 estrelas e mais de 25.000 garfos, com uma base de usuários incluindo MSNBC e NASA.

O Node.js é um ambiente de tempo de execução entre plataformas e uma biblioteca para executar aplicativos escritos em JavaScript fora do navegador (por exemplo, no servidor).

O Laravel é uma estrutura de aplicativo da Web PHP livre e de código aberto, projetada para o desenvolvimento de aplicativos da Web modelo – view – controller (MVC). O Laravel é lançado sob a licença do MIT, com seu código-fonte hospedado no GitHub.

O Microsoft Visual Studio é um ambiente de desenvolvimento integrado (IDE) da Microsoft. Ele é usado para desenvolver programas de computador para o Microsoft Windows, bem como sites da Web, aplicativos da Web e serviços da Web. O Visual Studio usa plataformas de desenvolvimento de software da Microsoft, como a API do Windows, o Windows Forms, o Windows Presentation Foundation, o Windows Store e o Microsoft Silverlight. Pode produzir código nativo e código gerenciado.

O WordPress é uma ferramenta de blogs gratuita e de código aberto e um sistema de gerenciamento de conteúdo (CMS) baseado em PHP e MySQL. Os recursos incluem uma arquitetura de plug-in e um sistema de modelos. O WordPress foi usado por mais de 22,0% dos 10 milhões de websites desde agosto de 2013. O GitHub é um serviço de hospedagem baseado na web do repositório Git que oferece todas as funcionalidades do Git, além de adicionar muitos recursos próprios. Ao contrário do Git, que é estritamente uma ferramenta de linha de comando, o Github fornece uma interface gráfica e de desktop baseada na web, bem como integração móvel.

Java é uma linguagem de programação de computador que é simultânea, baseada em classe, orientada a objetos e especificamente projetada para ter o menor número possível de dependências de implementação.
