    O .NET Framework (pronuncia-se dot net) é um framework de software desenvolvido pela Microsoft que é executado principalmente no Microsoft Windows. Ele inclui uma biblioteca de classes grande conhecida como FCL (Framework Class Library) e fornece interoperabilidade de idioma (cada idioma pode usar código escrito em outros idiomas) em várias linguagens de programação. Os programas criados para o .NET Framework são executados em um ambiente de software (em contraste com o ambiente de hardware), conhecido como Common Language Runtime (CLR), uma máquina virtual de aplicativos que fornece serviços como segurança, gerenciamento de memória e tratamento de exceções. FCL e CLR juntos constituem o .NET Framework.

    Na computação, o código-fonte é qualquer coleção de instruções de computador (possivelmente com comentários) escrita usando alguma linguagem de computador legível, geralmente como texto. O código-fonte de um programa é especialmente projetado para facilitar o trabalho de programadores de computador, que especificam as ações a serem executadas por um computador, principalmente escrevendo código-fonte. O código-fonte é frequentemente transformado por um programa compilador em código de máquina de baixo nível, entendido pelo computador. O código da máquina pode então ser armazenado para execução em um momento posterior. Alternativamente, um intérprete pode ser usado para analisar e executar os resultados do programa de código-fonte diretamente na hora.
